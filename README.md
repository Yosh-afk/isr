# Calculadora ISR 2026

Aplicacion web estatica para estimar el ISR de sueldos y salarios en Mexico con periodo mensual o quincenal.

## Archivos

- `index.html`: estructura de la app.
- `styles.css`: diseno visual responsivo.
- `app.js`: tarifas 2026, subsidio para el empleo y calculo.

## Como usarla

Abre `index.html` en el navegador. No requiere servidor, base de datos ni instalacion de dependencias.

## Publicar gratis en GitHub Pages

1. Crea un repositorio en GitHub, por ejemplo `calculadora-isr`.
2. Sube estos archivos al repositorio.
3. Entra a `Settings > Pages`.
4. En `Build and deployment`, selecciona `Deploy from a branch`.
5. Elige la rama `main` y la carpeta `/root`.
6. Guarda los cambios.

GitHub publicara la app en una URL parecida a:

```text
https://tu-usuario.github.io/calculadora-isr/
```

## Fuentes de calculo

- Anexo 8 de la Resolucion Miscelanea Fiscal para 2026, tarifas aplicables a retenciones de ISR.
- Decreto publicado el 31 de diciembre de 2025 que modifica el subsidio para el empleo.
- UMA mensual 2026 publicada por INEGI.

Esta herramienta es informativa y no sustituye asesoria fiscal ni un sistema formal de nomina.
